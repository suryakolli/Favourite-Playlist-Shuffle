Section 2 – Project Description and why you choose this project

This is a basic API integration project, API's are taken from Spotify and GeniusAPI. This project intends to save one's favourite spotify playlists and shuffle the random songs everytime the page is reloaded.

Section 3 – Description of technology used for your site

Front End: ReactJS
Back End: Python-Flask
Database: PostgreSQL
Deployment Framework: Heroku
Other API: SpotifyAPI, GeniusAPI
